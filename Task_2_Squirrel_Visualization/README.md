# 🐿️ Central Park Squirrel Census Data Visualization (2018)

This project visualizes and explores data from the **2018 Central Park Squirrel Census**, focusing on demographics, behavior, and location-based trends of squirrels in New York City’s Central Park. The analysis uses Python libraries such as Pandas, Matplotlib, and Seaborn to generate insightful visualizations and statistics.

## 📦 Dataset

- **Source:** NYC Open Data
- **Filename:** `2018_Central_Park_Squirrel_Census_-_Squirrel_Data_20250526.csv`
- **Key Features:** Location (X, Y), Age, Primary Fur Color, Shift (AM/PM), and Hectare Squirrel Number

## 🔍 Features & Visualizations

- **Missing Values Summary**
- **Age Distribution (Bar Plot)**
- **Primary Fur Color Distribution (Bar Plot)**
- **Squirrel Count in Hectares (Box Plot)**
- **Correlation Heatmap (Numeric Fields)**
- **Pair Plot of Location and Count**
- **Pie Chart of Sighting Shifts (AM vs PM)**
- **Fur Color Distribution by Shift (Count Plot)**
- **Location Map of Sightings (X vs Y Scatter Plot)**

## 🛠️ Libraries Used

- `pandas`
- `matplotlib`
- `seaborn`
